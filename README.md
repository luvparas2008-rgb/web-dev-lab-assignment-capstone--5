# Online Learning Platform – Capstone Project (Lab Assignment 5)

This is my capstone project for Lab Assignment 5. I chose the **Online Learning Platform** theme. The website is fully made with **HTML5 and CSS3**, and it has a clean and responsive layout.

##  Features Included
- Header with navigation links  
- Hero section with banner image and CTA  
- Course cards made using CSS Grid  
- Testimonials section  
- Pricing section with multiple plans  
- Contact form with labels and required fields  
- Fully responsive using Flexbox, Grid, and media queries  
- Hover effects, transitions, and consistent color theme  

##  Breakpoints Used
- **Desktop:** 1024px and above  
- **Tablet:** Below 768px  
- **Mobile:** Below 480–600px  

##  How to View the Project
1. Download the project folder  
2. Open the file **index.html** in any browser  

##  Files Included
- index.html  
- style.css  
- images/ (all images used in the site)  
- README.md  

This project follows semantic HTML structure and meets all requirements of the assignment.
